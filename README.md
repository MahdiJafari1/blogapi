
# Blog API

A fully functional yet simple API with documentation for blog.


## API Reference

#### Get all posts

```http
  GET /api/v1/
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**.  |

#### Get post

```http
  GET /api/v1/<int:pk>
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `int` | **Required**. Id of item to fetch |




## API Documentation

```http
  GET /api/v1/swagger/
```

```http
  GET /api/v1/redoc/
```
